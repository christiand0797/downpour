## 🎯 **USAGE EXAMPLES**

### **✅ Integration Examples**

**Basic Integration**:
```python
from enhanced_hardware_integration import initialize_enhanced_hardware

# Initialize enhanced hardware monitoring
hardware = initialize_enhanced_hardware()

# Start monitoring
hardware.start_monitoring()

# Get current metrics
metrics = hardware.get_current_metrics()
print(f"CPU: {metrics.cpu_percent:.1f}%")
print(f"Memory: {metrics.memory_percent:.1f}%")
print(f"Temperature: {metrics.cpu_temperature:.1f}°C")
```

**Gauge Integration**:
```python
# Get gauge data for CPU
cpu_gauge = hardware.get_responsive_gauge_data("cpu")
print(f"CPU Gauge: {cpu_gauge['value']:.1f}% ({cpu_gauge['status']})")
print(f"Color: {cpu_gauge['color']}")
print(f"Trend: {cpu_gauge['trend']}")

# Get gauge data for temperature
temp_gauge = hardware.get_responsive_gauge_data("temperature")
print(f"Temp Gauge: {temp_gauge['value']:.1f}°C ({temp_gauge['status']})")
```

**Callback Integration**:
```python
def on_hardware_update(metrics):
    """Handle hardware updates"""
    if metrics.cpu_percent > 80:
        print("⚠️ High CPU usage detected!")
    if metrics.cpu_temperature > 75:
        print("🌡️ High temperature detected!")

# Register callback
hardware.add_callback(on_hardware_update)
```

**GUI Integration**:
```python
import tkinter as tk

# Create gauge system
root = tk.Tk()
gauge_system = hardware.create_gauge_system(root)

# Run the GUI
gauge_system.run()
```
