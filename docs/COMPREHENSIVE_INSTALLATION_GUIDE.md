## 🚨 **TROUBLESHOOTING**

### **⚠️ Common Installation Issues**

**Issue 1: "Some features failed to install"**
- **Cause**: Network issues or incompatible Python version
- **Solution**: Check internet connection, try again as admin
- **Workaround**: Manual installation of failed packages

**Issue 2: "tensorflow installation failed"**
- **Cause**: Incompatible system architecture or Python version
- **Solution**: Use CPU-only version or skip ML features
- **Workaround**: Install tensorflow-cpu instead

**Issue 3: "pillow-heif installation failed"**
- **Cause**: Missing system libraries
- **Solution**: Install Visual C++ redistributable
- **Workaround**: Skip HEIF support, use regular Pillow

**Issue 4: "selenium installation failed"**
- **Cause**: Missing browser drivers
- **Solution**: Install Chrome/Firefox drivers
- **Workaround**: Skip web automation features
