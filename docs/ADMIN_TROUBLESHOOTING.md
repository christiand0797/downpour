## 🔧 **ADMIN ACCESS TROUBLESHOOTING**

### **⚠️ Common Admin Issues & Solutions**

**Issue 1: "No UAC prompt appears"**
- **Cause**: Launcher doesn't request admin privileges
- **Solution**: ✅ **FIXED** - All launchers now request admin
- **Action**: Use updated launchers

**Issue 2: "UAC prompt appears but fails"**
- **Cause**: User cancels UAC or insufficient permissions
- **Solution**: Approve UAC prompt with admin account
- **Action**: Click "Yes" on UAC prompt

**Issue 3: "Python not found"**
- **Cause**: Python not installed or not in PATH
- **Solution**: Install Python 3.8+ and add to PATH
- **Action**: Download from https://www.python.org/downloads/

**Issue 4: "Access denied" after UAC**
- **Cause**: Limited user account or group policy
- **Solution**: Use administrator account
- **Action**: Right-click → "Run as administrator"

**Issue 5: "Windows Defender blocks"**
- **Cause**: Windows Defender blocks execution
- **Solution**: Add folder to Windows Defender exclusions
- **Action**: Allow in Windows Security

## 🚨 **ADVANCED TROUBLESHOOTING**

### **⚠️ Advanced Admin Issues**

**Issue 1: Group Policy Blocks Admin**
- **Symptoms**: UAC prompt blocked or limited
- **Solution**: Contact system administrator
- **Workaround**: Use non-admin launcher with limited features

**Issue 2: Corporate Security Blocks**
- **Symptoms**: Execution blocked by corporate security
- **Solution**: Add to corporate allowlist
- **Workaround**: Use portable Python version

**Issue 3: Python Path Issues**
- **Symptoms**: Python not found even when installed
- **Solution**: Add Python to system PATH
- **Workaround**: Use full Python path in launcher

**Issue 4: Windows Defender Aggressive**
- **Symptoms**: Launcher blocked by Windows Defender
- **Solution**: Add folder to exclusions
- **Workaround**: Temporarily disable real-time protection

**Issue 5: System File Corruption**
- **Symptoms**: UAC system files corrupted
- **Solution**: Run system file checker (sfc /scannow)
- **Workaround**: Use system restore point
